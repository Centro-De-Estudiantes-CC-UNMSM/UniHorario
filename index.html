Manual de uso

  • Para cada sección de un curso debe tener exactamente el mismo nombre.
  • Las horas deben escribirse en formato decimal: 9:30 como 9.50 y 9:45 como 9.75.
  • Si un curso en una sección tiene múltiples horarios, se deben separar los días con comas (ej. lunes, martes) y las horas correspondientes también con comas (ej. 9-12.50, 10-11).
